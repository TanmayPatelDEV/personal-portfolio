# Personal Developer Portfolio

A fully responsive, modern personal developer portfolio built from scratch. This project features a clean dark-mode user interface, smooth hover animations, and a flexible layout designed to look great on desktop and mobile devices. Built as part of the Web Development Internship at Intern Alpha.

## 🔗 Live Demo
👉 [https://tanmay-patel-swe.netlify.app/]

## 🚀 Key Features Implemented
- **Modern Dark-Mode UI:** Designed with a sleek, dark aesthetic to highlight project cards and technical skills without causing eye strain.
- **100% Responsive Design:** Implemented CSS Flexbox to ensure all project grids and text content scale perfectly down to mobile screens.
- **Interactive Elements:** Added CSS transitions for hover states on buttons and project cards to improve user experience.
- **Clean Architecture:** Wrote semantic HTML5 and organized CSS3 for maximum readability and future scalability.

## 🛠️ Tech Stack Used
- **HTML5:** Semantic structure and content organization.
- **CSS3:** Custom styling, Flexbox layout, and animation transitions.
